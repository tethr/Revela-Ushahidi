== Description ==
This plugin enhances the standard map info-bubbles. It will display the incident description, lat/lon, time, location,author verified state and news sources. It also introduces a tabbed architecture for images and customform data.

== Installation ==
1. Copy the entire /InfoWindow/ directory into your /plugins/ directory.
2. Activate the plugin.
